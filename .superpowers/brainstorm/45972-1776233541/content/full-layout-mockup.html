All Pane Types

Available from the Views menu in the session toolbar

Claude Terminal

Primary Claude Code session. Always present. xterm.js with statusline integration.

Shortcut: always visible

Partner Terminal

Companion shell for dev servers, tests, builds. Existing feature, now a draggable pane.

Shortcut: existing toggle

Diff Viewer

Auto-populated via git diff when files change. File list + inline diff with line comments.

Shortcut: Ctrl+Shift+D

Preview Pane

Embedded webview for dev servers, HTML, PDF, images. Auto-detects localhost URLs in terminal output.

Shortcut: Ctrl+Shift+P

File Editor

Monaco-based editor. Click file paths in chat/diff to open. Save, discard, disk-change warnings.

Shortcut: click file path

Side Chat

Branched conversation overlay. Full PTY session. Reads main context, doesn't write back. Works over SSH.

Shortcut: Ctrl+;
